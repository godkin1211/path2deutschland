更新GitHub Pages

本後台系統會直接修改data/目錄下的JSON檔案，請使用以下Git指令來更新GitHub Pages：

git add data/
git commit -m "更新網站內容"
git push

使用指南：

  1. 啟動後台伺服器：cd admin && npm start
  2. 在這個管理面板中新增/刪除內容
  3. 所有更改會直接寫入data/目錄下的JSON檔案
  4. 使用git push來更新GitHub Pages

提示： 後台伺服器運行在 localhost:3001，需要Node.js環境。
