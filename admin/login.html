GitHub Token 認證

請輸入您的 GitHub Personal Access Token

需要 repo 和 user:email 權限

如何獲取 GitHub Token？
  1. 登入 GitHub 並前往 Settings → Developer settings → Personal access tokens → Tokens (classic)
  2. 點擊 "Generate new token" → "Generate new token (classic)"
  3. 選擇適當的過期時間
  4. 勾選以下權限：
    • repo - 完整的repository控制權
    • user:email - 讀取用戶email
  5. 點擊 "Generate token" 並複製生成的token
